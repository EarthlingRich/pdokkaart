Help en (ontwikkel) Documentatie

Om u op weg te helpen met PDOKKaart of de PDOKKaart wizard zijn er de volgende mogelijkheden:

Ga naar de documentatie pagina's. De rst sources hiervan staan in de docs directory.

U kunt ook de voorbeelden bekijken in de examples directory.

Op het forum deel van www.pdok.nl is een apart deel ingericht voor pdok kaart onderwerpen.

De code staat in Github.

Bugs of feature requests, worden ook op Github bijgehouden in de Github issues lijst.
