De API

De PDOKKaart API is een manier waarop PDOK kaartmateriaal op een betrekkelijk eenvoudige wijze te gebruiken is in eigen websites. Hierbij is het enerszijds mogelijk via url-parameters een kaart te maken welke in een website is op te nemen. Anderszijds is het mogelijk de API op te nemen in een eigen HTML pagina en middels JavaScript de API te besturen en te gebruiken. Als derde mogelijkheid is de Wizard gemaakt waarmee een URL met parameters te maken is of een aanzet te maken is voor een eigen HTML pagina. - mogelijkheden
- onmogelijkheden
+ omdat JavaScript gebruikt wordt en er geen alternatief bestaat zonder JavaScript is het niet rijkswebrichtlijn compliant. + vanwege in-browser veiligheidsrestricties is het niet mogelijk data welke op een externe server rechtstreeks te benaderen, om deze onmogelijkheid te omzeilen is een proxy script op de server gezet welke slechts bekende URL's accepteert (bijvoorbeeld maps.google.nl). Deze lijst met bekende URL's kan uitgebreidt worden. Wendt U hiervoor tot Geonovum.
