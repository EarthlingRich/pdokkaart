Class Detail
Pdok.Api(config)
Creates an instance of the Pdok Api based on an optional config object more ...

If there is not a config object, all properties of the api have a default value.

The config object is a hash object like: 
{ loc:'150000,380000', div:'map' }

The OpenLayers.Util.getParamaters() function returns such an object. Giving the possiblitiy to create an Api object which uses url params in just two lines of code:

   var config = OpenLayers.Util.getParameters();
   api = Pdok.Api(config);
All properties of the config object are overriding default values of Api
Parameters:
{Object} config
A hash object with all possible config key/value pairs

        // all point marker styles will use mt0 as default 
        // so you only have to define the props that are different from mt0 
        // mt0, pt0, lt0 are defined in pdok-api.js, so defining it here will override that one 
    { 
        id: 'mt1', 
        name: 'Informatiebord blauw', 
        externalGraphic: Pdok.ApiUrl + '/markertypes/emblem-notice.png', 
        graphicHeight: 32, 
        graphicWidth: 32, 
        graphicYOffset: -16 
    }, 
    {
        id: 'pt18', 
        fillOpacity: 0.0, 
        strokeColor: 'orange', 
        strokeWidth: 2, 
        name: 'Oranje'
    },
    {
        id:'lt8',  
        strokeColor:'green',  
        strokeWidth:3,  
        strokeOpacity:0.5,  
        name:'groen 2px transparant'  
    }
